# MERN Фотогалерея

## Возможности

 - Регистрация и авторизация
 - Настройка профиля
 - Загрузка изображений
 - Поиск и просмотр профилей других пользователей

 - Redux Toolkit
 - React Router Dom
 - Formik
 - Bootstrap
 - React-Bootstrap
 - SCSS

### Бэк-энд
 - NodeJS
 - Express
 - Mongoose
